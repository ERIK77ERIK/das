2.median-ն առաջնահերթություն չի և բրաուզերը կարդում է ներքեվից վերև
3.եթե նույն media-ին երկու արժեք ենք տալի բրաուզերը ընտրում է հատումը
@media screen and (max-width:600px) and (min-width:300px) {
selector {
}
}
4.եթե երկու իրար տակ media է գրած media-ների width-երի րնդհանուր մասի ժամանակ բրաուզերը ընտրում է տարբեր media-ների մեջի նույն selector-ի մեջի գրած min-width-ի և max-width-ի 2 պայմաներն ել առաջնահերթ համարելով մեծ արժեքը
օրինակ եթե նույն selector-ի մեջի min-width-ը 200px է և max-width-ը 100px ապա 200-ից ավել չի փոքրացնում բայց չի ել մեծացնում
