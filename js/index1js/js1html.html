JAVASCRIPT - ՆԵՐԱԾՈՒԹՅՈՒՆ



CSS 15

1.Javascript-ը ծրագրաորման լեզու է որը չեն հասկանում համակարգիչները, համակարգիչները հասկանում են միայն 0-ներ և 1-եր, որը համակարգչային լեզու է, երբ համակարգիչները նոր էին ստեղծվել հենց այդպես էին ծրագրաորում(0-ներով և 1-երով), հետո ստեղծվեցին ավելի հեշտ լեզուներ և ծրագիրներ որը այդ լեզվից կթարգմանեն դեպի համակարգչային լեզու, այդ լեզուներից մեկը Javascript-ն է որը ունի համապատասխան ուրիշ ծրագիր, որը իրեն կթարգմանի համակարգչային լեզվի և կաշխատացնի, ամեն բրաուզեր ունի մի այսպիսի ծրագիր արդեն իր մեջ
2.ծրագրաորման լեզուների մեջ գոյություն ունեն հրամաններ, հրամանները ունեն անուններ և ամեն հրամանը ինչ-որ գործողություն է կատարում (մեր օրերում հրաման չենք կոչում այլ ֆունկցիա)
3.ֆունկցիան մենք կանչում(աշխատացնում) ենք անունի կողքը դնելով փակագիծներ
4.Javascript-ում երբ որ մենք դադարում ենք ինչ-որ բան ասել վերջում պետքա վերջակետ դնենք բայց Javascript-ում դա վերջակետ չի այլ կետ ստորակետ( ; )
5.alert();-ը մի հրաման է ում դերը պատուհան բացելն է(alert-ը դա ֆունկցիա է, որի անունը alert է)
6.մաթեմաթիկական բոլոր օրենքներին հետևում է Javascript-ը
alert(6+4*2); կտպի 14
alert((6+4)*2); կտպի 20
7.տեքստը պետք է գրենք երկու կամ մեկ ստորակետների մեջ
alert("77" + '33'); կտպի 7733

alert('բարև' + 77); կտպի բարև77
alert("բարև " + 77); կտպի բարև 77
alert("բարև" + ' ձեզ'); կտպի բարև ձեզ

alert("բարև" + "" + "ձեզ"); կտպի բարևձեզ
alert("բարև" + " " + "ձեզ"); կտպի բարև ձեզ

alert(7 + 5 + "բարև" + 7 + 5); կտպի 12բարև75
