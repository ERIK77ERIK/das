ORDER(կարգ)



FLEX 2

1.order-ը թույլ է տալիս child էլեմենտները տեղափոխել, և փոխել հերթականությունը
2.order-ը մենք տալիս ենք երեխա էլեմենտներին
3.էլեմենտների սկզբնական order-ի արժեքը order:0; է
4.էլեմենտները իրար կողք դասաորվում են order-ի փոքր արժեքից մեծ արժեք ձախից աջ հերթականությամբ,իսկ եթե երկու էլեմենտի order-ին տալիսենք նույն արժեքը բրաուզերը դասաորում է սկզբնական դասաորվածությամբ
.child:nth-child(1) {
order:1;
}
.child:nth-child(2) {
order:1;
background:blue;
}
.child:nth-child(3) {
order:-1;
}
.child:nth-child(4) {

background: red;
}
